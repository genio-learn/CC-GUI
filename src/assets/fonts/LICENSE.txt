MesloLGS NF
===========

The bundled MesloLGSNF-Regular.woff2 and MesloLGSNF-Bold.woff2 are the
MesloLGS NF fonts, a Nerd Fonts patched variant of Meslo LG (itself derived
from Apple's Menlo). They are bundled so the embedded terminal can render the
Claude Code status line's icon glyphs regardless of which fonts the user has
installed.
