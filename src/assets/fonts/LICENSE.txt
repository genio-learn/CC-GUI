MesloLGS NF
===========

The bundled MesloLGSNF-Regular.woff2 and MesloLGSNF-Bold.woff2 are the
MesloLGS NF fonts, a Nerd Fonts patched variant of Meslo LG (itself derived
from Apple's Menlo). They are bundled so the embedded terminal can render the
Claude Code status line's icon glyphs regardless of which fonts the user has
installed.

Source:  https://github.com/romkatv/powerlevel10k#meslo-nerd-font-patched-for-powerlevel10k

IBM Plex
========

The bundled IBMPlexSans-*.woff2 and IBMPlexMono-*.woff2 are the IBM Plex Sans
and IBM Plex Mono fonts, used for the app chrome (UI type) and monospace chrome
(diffs, counters, badges) respectively. They are self-hosted so the interface
renders identically regardless of which fonts the user has installed.

Source:  https://github.com/IBM/plex
